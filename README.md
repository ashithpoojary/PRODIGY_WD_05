## PRODIGY_WD_05
Weather Web Application

## Description
A modern weather web application developed using HTML, CSS, and JavaScript that provides real-time weather updates using the OpenWeather API. Users can search any city to view temperature, weather conditions, humidity, wind speed, and dynamic weather icons through a clean, responsive, and user-friendly interface.

## Features
- Real-Time Weather Data
- Search Weather by City Name
- Dynamic Weather Icons
- Temperature Display
- Humidity Information
- Wind Speed Information
- Responsive Web Design
- Interactive User Interface
- Modern Weather App Design

## Technologies Used
- HTML
- CSS
- JavaScript
- OpenWeather API
